Theoretical Basis

This engine demonstrates the strict computational equivalence between Context-Free Grammars (CFGs) and Pushdown Automata (PDAs). Any language generated by a CFG can be recognized by a PDA, and vice-versa.

CFG → PDA (Top-Down Parsing)

Creates a 3-state PDA that non-deterministically simulates a leftmost derivation. It pushes grammar rules to the stack and matches terminals against the input.

PDA → CFG (Triplet Construction)

A complex reduction that maps internal path transitions. A variable A[p,q] generates all strings that navigate the machine from state p to q ending with an empty stack.

Context-Free Grammar (CFG)

G = (V, Σ, R, S)

  • V : Finite set of Variables (Non-terminals).
  • Σ : Finite set of Terminals (Input alphabet).
  • R : Production Rules defining language syntax.
  • S : Start Variable representing the root.

Pushdown Automaton (PDA)

P = (Q, Σ, Γ, δ, q₀, F)

  • Q : Finite set of internal states.
  • Γ : Stack Alphabet (memory structure).
  • δ : Transition Function determining moves.
  • F : Set of Accept States for valid strings.

Reduction Algorithm Steps (CFG to PDA)

1 Structuring states
Create start, loop, and accept states.
2 Stack Initialization
Push $ and Start variable S.
3 Rule Expansion
Substitute variables via ε-transitions.
4 Terminal Matching
Pop terminals matching input characters.
5 Accept Mode
Accept on an empty stack ($).

Canonical Example Cases

1. The aⁿbⁿ Language
S → aSb | ε

A classic mathematical example proving memory is needed. The PDA pushes 'a's to the stack to keep count, then pops them to ensure an exact matching number of 'b's follow.

2. Palindromes (w wᴿ)
S → aSa | bSb | a | b | ε

Demonstrates non-determinism. The machine has no way to know when the "middle" of the string occurs, so it must branch its computation to "guess" the midpoint dynamically.

3. Balanced Parentheses
S → SS | (S) | ε

The foundation of compiler syntax analysis. PDAs perfectly track nested structures (like code blocks or equations) by pushing on open brackets and popping on closing brackets.
